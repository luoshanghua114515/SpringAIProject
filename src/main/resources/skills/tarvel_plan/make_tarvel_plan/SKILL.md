## Execution Flow
1. 使用搜索引擎搜索相关内容
2. 图片内容无需下载，用http的url展示即可
3. 结合相关内容生成.md文件，如果用户指定要求pdf则生成pdf
4. 生成的文件一定要把链接展示给用户


## Examples
- 上海3天旅游
- 北京出差2天
- 杭州周末游
